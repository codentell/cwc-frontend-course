## Overview

This module, students will be introduced to Bootstrap. Everything they have learned up until now will assist them as they navigate the use of Bootstrap.

* Design thinking
  * Empathy
  * Definition
  * Ideation
  * Prototyping
  * Testing

* Bootstrap
  * Bootstrap documentation
  * Bootstrap HTML/CSS
  * Bootstrap components
  * Bootstrap grid
